🧮 BMI Calculator (CLI)

A simple command-line Body Mass Index (BMI) calculator written in Python. This tool takes user input for weight and height, calculates BMI, and displays the corresponding health category.
📌 Features
Calculate BMI using standard formula
Categorizes BMI into:
Underweight
Normal weight
Overweight
Obese
Input validation for positive numbers
Handles invalid input gracefully
📂 Project Structure
bmi-calculator/
│── bmi_calculator.py
│── README.md
Code Overview
calculate_bmi(weight, height) → Returns BMI value
get_category(bmi) → Determines BMI category
main() → Handles user input and output
