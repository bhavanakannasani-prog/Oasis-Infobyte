🔐 Password Generator (CLI)
A simple and customizable command-line password generator built with Python. Generate secure passwords based on your preferred length and character types.
📌 Features
🔢 Custom password length
🔤 Option to include letters
🔐 Option to include digits
💥 Option to include symbols
⚡ Fast and lightweight CLI tool
❌ Input validation and error handling
📂 Project Structure
password-generator/
│── password_generator.py
│── README.md
🛠️ How It Works
User inputs desired password length
Selects character types:
Letters (A–Z, a–z)
Digits (0–9)
Symbols (!, @, #, etc.)
Program combines selected character sets
Generates a random password using Python’s random module
