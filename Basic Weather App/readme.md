🌦️ Weather App (CLI)
A simple command-line weather application built with Python that fetches real-time weather data using the Open-Meteo API — no API key required!
📌 Features
🌍 Get weather by city name
📍 Automatically fetches latitude & longitude
🌡 Displays current temperature
🌬 Shows wind speed and direction
⏰ Displays observation time
❌ Handles invalid city input gracefully
🔑 No API key required
📂 Project Structure
weather-app/
│── weather_app.py
│── README.md
🛠️ How It Works
User enters a city name
App fetches coordinates using Open-Meteo Geocoding API
Uses coordinates to get current weather data
Displays formatted output in terminal
🌐 APIs Used
Open-Meteo Geocoding API
Open-Meteo Weather API
